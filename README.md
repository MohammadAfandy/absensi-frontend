# Absensi Frontend

Absensi app with React native and Expo.

### Expo Published App

```
https://expo.io/@pandi98/absensi
```

### Usage

#### Run the app

```
npm run start
```

or

```
yarn run start
```

or

```
expo start
```

#### Develop the app for iOS

```
npm run ios
```

or

```
yarn run ios
```

#### Develop the app for Android

```
npm run android
```

or

```
yarn run android
```
